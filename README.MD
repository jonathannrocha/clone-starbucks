
<h2 >"Clone do site da Starbucks"</h1>

Projeto desenvolvido com o intuito de aprender novas técnicas de Html e CSS3 copiando sites já prontos e com uma complexidade um pouco maior.

Foi possível treinar flexbox, grid, mobile firts, transition e transform.
